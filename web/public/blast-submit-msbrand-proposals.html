Proposal C — Input / Runtime Split Workspace

The 7 sections fall into two natural groups: Input (program, query, database, taxon) and Runtime (task profile, execution profile, algorithm parameters). This proposal mirrors that mental model with a 50/50 two-column workspace — left in Microsoft Blue, right in Microsoft Purple — and a floating pill-shaped Run bar at the bottom.
